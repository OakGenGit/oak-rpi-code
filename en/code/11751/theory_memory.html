Memory and Storage

Contents of the topic include:

Primary Storage

Why do computers have primary storage?

Primary storage is the computer’s main memory, which is directly accessible by the CPU and often much faster than secondary storage. Primary storage consists of:

  • RAM (Random Access Memory) – Temporary, volatile memory that stores data currently being used by the system.
  • ROM (Read-Only Memory) – Non-volatile memory that contains essential instructions, such as the computer’s boot process.

Secondary Storage

Why is secondary storage needed?

Unlike primary storage, secondary storage is used to store data long-term, even when the computer is turned off. Types of secondary storage include:

  • HDDs (Hard Disk Drives) – Traditional magnetic storage, commonly used in computers.
  • SSDs (Solid State Drives) – Faster, more durable storage with no moving parts.
  • Optical Discs – CDs, DVDs, and Blu-rays for storing media and software.
  • Flash Storage – USB drives and memory cards for portable data storage.

Units

How is data measured?

Computers store data in binary (1s and 0s). Storage capacity is measured in units such as:

  • Bit – The smallest unit of data (0 or 1).
  • Byte – 8 bits.
  • Kilobyte (KB) – 1,024 bytes.
  • Megabyte (MB) – 1,024 KB.
  • Gigabyte (GB) – 1,024 MB.
  • Terabyte (TB) – 1,024 GB.

Data Storage

How do different types of data get stored?

Computers store different types of data using various file formats:

  • Text files (.txt, .docx) – Store written information.
  • Image files (.jpg, .png, .gif) – Store pictures and graphics.
  • Audio files (.mp3, .wav) – Store sound recordings and music.
  • Video files (.mp4, .avi) – Store moving images and sound.
  • Executable files (.exe) – Contain instructions for running programs.

Compression

Why do we use compression?

Compression reduces the size of files so they take up less storage space and are quicker to transfer. There are two main types:

  • Lossy Compression – Removes some data to make the file smaller. Used in images (JPEG), audio (MP3), and video (MP4).
  • Lossless Compression – Reduces file size without losing data. Used in text files (ZIP) and images (PNG).
